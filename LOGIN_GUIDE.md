# 🔐 登录系统使用指南

本工具集已启用登录保护，仅供授权用户使用。

## 📋 默认账号

| 用户名 | 密码 | 说明 |
|--------|------|------|
| **admin** | admin123 | 管理员账号 |
| **eyou** | eyou2026 | 普通账号 |

## 🚀 快速开始

1. **访问登录页面**
   - 本地：打开 `login.html`
   - 在线：访问 `https://你的域名/login.html`

2. **输入账号密码**
   - 输入上述默认账号之一
   - 勾选"记住我"可保持7天登录状态

3. **开始使用**
   - 登录成功后自动跳转到工具首页
   - 可使用所有工具功能

## 🔧 修改密码

### 方法1：直接修改配置文件（推荐）

编辑 `assets/js/auth.js` 文件：

```javascript
config: {
    users: [
        { username: 'admin', password: '你的新密码' },
        { username: 'eyou', password: '你的新密码' },
        // 添加更多用户
        { username: '新用户名', password: '新密码' }
    ],
    // ...
}
```

### 方法2：添加新用户

在 `users` 数组中添加新的用户对象：

```javascript
users: [
    { username: 'admin', password: 'admin123' },
    { username: 'eyou', password: 'eyou2026' },
    { username: 'user1', password: 'pass123' },    // 新用户1
    { username: 'user2', password: 'pass456' }     // 新用户2
]
```

## 🛡️ 安全建议

### ⚠️ 重要提示

1. **立即修改默认密码**
   - 默认密码 `admin123` 和 `eyou2026` 非常简单
   - 部署到线上前务必修改

2. **使用强密码**
   - 至少8位字符
   - 包含大小写字母、数字、特殊字符
   - 例如：`Abc@1234`, `Eyou#2026`

3. **定期更换密码**
   - 建议每3个月更换一次
   - 不要与其他网站使用相同密码

### 🔒 更安全的方案

当前的认证系统适用于私人使用，如果需要更高的安全性：

1. **使用后端认证**
   - 将认证逻辑移到服务器端
   - 使用加密算法存储密码（如 bcrypt）

2. **使用第三方认证**
   - Google OAuth
   - GitHub OAuth
   - 企业微信/钉钉认证

3. **添加双因素认证（2FA）**
   - 短信验证码
   - 邮箱验证码
   - Google Authenticator

## 💡 常见问题

### Q: 忘记密码怎么办？
**A:** 直接修改 `assets/js/auth.js` 文件中的密码即可。

### Q: 登录后多久会过期？
**A:**
- 勾选"记住我"：7天后过期
- 不勾选：1天后过期

### Q: 如何退出登录？
**A:** 点击首页右上角的"退出"按钮。

### Q: 可以同时登录多个账号吗？
**A:** 不可以，一个浏览器只能保持一个账号的登录状态。

### Q: 清除浏览器缓存会退出登录吗？
**A:** 是的，登录信息保存在 localStorage 中，清除浏览器数据会导致退出登录。

### Q: 如何禁用登录保护？
**A:** 不建议禁用。如果确实需要，可以删除各页面中的以下代码：
```javascript
<script src="../../assets/js/auth.js"></script>
<script>
    Auth.requireLogin();
</script>
```

## 🔄 登录状态管理

### 自动延长会话
登录状态在使用过程中会自动刷新，无需重复登录。

### 手动延长会话
在浏览器控制台执行：
```javascript
Auth.extendSession(30); // 延长30天
```

### 检查登录状态
在浏览器控制台执行：
```javascript
Auth.isLoggedIn();        // 返回 true 或 false
Auth.getCurrentUser();    // 返回当前用户信息
```
